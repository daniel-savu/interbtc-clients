MockBitcoinCore

A simulated bitcoin-core interface. It combines the roles of bitcoin-core and the staked relayer: it automatically relays the generated transactions to the parachain. It does the minimum amount of work it can get away with, and the relayed data may be technically invalid. For example, all generated transactions share the same dummy input uxto.
